Website and Map created by: Adam Bourque

3D Interactive Map was created in QGIS using plugin, Qgis2threejs. Hillshade and DEM data for 3D map were obtained from KYGeoportal Image Services, and clipped to Floracliff border using ArcGIS 10.3.1. Trail data was obtained directly from Floracliff, and was converted from KMZ to KML using Google Earth to be compatible with 3D Interactive Map. Trail map and land management zone map were created using CalTopo and used AvenzaMap for onsite mapping.
